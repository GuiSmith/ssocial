Como instalar e usar o projeto SSocial

Já instalou?

Pode acessar o projeto através da seguinte URL:

localhost/projects/ssocial/main/index.php

Instalando PHP

Para instalar o PHP no Linux Ubuntu, execute os comandos abaixo no terminal:

sudo apt update
sudo apt install php

Instalando SQLite

Para instalar o SQLite no Linux Ubuntu, execute os comandos abaixo no terminal:

sudo apt update
sudo apt install sqlite3

Instalando extensão do SQLite com PHP

Para se conectar aos bancos de dados SQLite, é necessário instalar a extensão específica do PHP.

Primeiro, verifique a versão do PHP instalada com o comando:

php -v

Depois, instale a extensão correspondente à sua versão do PHP. Por exemplo, para PHP 7.4:

sudo apt install php7.4-sqlite3

Reinicie o servidor Apache para aplicar as alterações:

sudo systemctl restart apache2

Instalando git

Para garantir que você tenha o Git instalado, use os comandos abaixo:

sudo apt update
sudo apt install git

Clonando o repositório do GitHub

Para usar o projeto, navegue até a pasta HTML no Linux Ubuntu:

cd /var/www/html

Crie uma pasta chamada “projects”:

mkdir projects

Acesse a pasta "projects":

cd projects

Clone o repositório do GitHub:

git clone https://github.com/GuiSmith/ssocial.git

Pronto!

Agora você pode acessar o projeto através da seguinte URL:

localhost/projects/ssocial/main/index.php

Possíveis problemas

Se encontrar um problema onde o sistema diz que o banco está em modo somente leitura, ajuste as permissões do diretório no terminal:

cd /var/www/html/projects/
sudo chmod 777 -R ssocial
